STACK

Stack is a linear data structure which is used to store and retrive data items using the principle "First In Last Out" (or) "Last In First Out"

Methods used to organise Stack are :

  • Push()

    • This method is used to push data items into the stack in a linear fashion (side-by-side). Recently pushed items will be at top of stack

  • Pop()

    • This method is used to pop data items from the stack. Each time it is called the top item will be popped out (or) Last pushed item will be popped out

  • Display()

    • This method is used to display data items present in the stack. The items are diplayed from top of stack to the bottom of stack

We can implement stack in two ways

  • Static implementation : Here the stack size is fixed i.e., size of stack cannot be changed during run time or execution of the program
  • Dynamic Implementation : Here the size of stack is dynamic i.e., each time we use push method a new memory from heap is attached to the existing stack
